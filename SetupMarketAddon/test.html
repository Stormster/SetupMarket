Setup Market - Content Manager Integration Test

This page demonstrates how the setupmarket:// URL protocol works with Content Manager.

Test Links

Click on the button below to test the integration. This should open Content Manager and prompt you to save the setup.

Add Setup to Content Manager

Note

For this test to work:

  1. The Setup Market addon must be installed in Content Manager
  2. The setupmarket:// URL protocol must be registered with Windows
  3. Setup ID 47596 must exist on setupmarket.net

HTML Code Example

<a href="setupmarket://setup/47596">Add to Content Manager</a>

Replace "47596" with the actual setup ID from your website.
